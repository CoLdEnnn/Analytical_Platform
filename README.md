# Analytical_Platform

### Overview
This project is a simple analytical dashboard built with Node.js, Express, MongoDB, and Chart.js.
It retrieves time-series data from MongoDB, allows filtering by date range and selected fields, and visualizes the results with charts and statistical metrics.

### Features
Filter measurements by date range
Select different fields for analysis:
- Temperature (Â°C)
- Humidity
- Pressure (millibars)
Time-series visualization using Chart.js
Statistical metrics:
- Average
- Minimum
- Maximum
- Standard Deviation
RESTful API built with Express
MongoDB aggregation for analytics
